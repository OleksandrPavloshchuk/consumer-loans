## back-app environment variables

- **SERVER_PORT** — HTTPS server port for back-app
- **DB_URL** — JDBC URL of the back-app database
- **DB_USERNAME** — database user
- **DB_PASSWORD** — database password
- **CAMUNDA_ADMIN_ID** — embedded Camunda admin id
- **CAMUNDA_ADMIN_PASSWORD** — embedded Camunda admin password
- **CAMUNDA_ADMIN_FIRST_NAME** — embedded Camunda admin first name
- **SSL_KEYSTORE_PATH** — absolute path to PKCS12 keystore
- **SSL_KEYSTORE_PASSWORD** — keystore password
- **JWT_SECRET** — predefined secret for signing JWT
